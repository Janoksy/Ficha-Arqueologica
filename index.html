Carga un ZIP para sincronizar fichas entre dispositivos del equipo.

💡 Flujo para sincronizar equipo:

1. Arqueólogo A exporta ZIP y lo comparte
2. Arqueólogo B importa el ZIP
3. Las fichas se combinan sin duplicar
4. Cada uno puede seguir agregando fichas nuevas
